# AnyStyle Converter

أداة واجهة رسومية (GUI) لتحويل المراجع الببليوغرافية إلى تنسيقات منظمة (JSON, BibTeX, RIS, CSL, XML) باستخدام محرك AnyStyle.

## المتطلبات
- نظام تشغيل Ubuntu.
- اتصال بالإنترنت (لعملية التثبيت فقط).

## خطوات التثبيت
1. قم بتحميل المستودع من GitHub:
   ```bash
   git clone [رابط_المستودع_الخاص_بك]
   cd anystyle-converter-main
   امنح صلاحيات التنفيذ لسكربت التثبيت والتشغيل:
    Bash

    chmod +x install.sh run.sh

    قم بتشغيل سكربت التثبيت (سيقوم بتثبيت كل المتطلبات تلقائياً):
    Bash

    ./install.sh

كيفية التشغيل

    من خلال قائمة التطبيقات: ابحث عن "AnyStyle Converter" في قائمة تطبيقات Ubuntu وافتحه بضغطة زر.

    من خلال الطرفية:
    Bash

    ./run.sh

الترخيص

تعتمد هذه الأداة على محرك AnyStyle الموزع تحت رخصة BSD-style license.
